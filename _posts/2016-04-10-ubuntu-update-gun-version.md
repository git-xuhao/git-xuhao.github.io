### 1：Gun Make 下载地址
```
ftp://ftp.gnu.org/gnu/make/
```
### 2： 下载并解压 make3.8.1.tar.gz

```
$ tar -zxvf make3.8.1.tar.gz
```

### 3： 编译Make

```
$ ./configure
$ make
```

### 4： 安装Make

```
$ sudo make install
```
